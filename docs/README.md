# Documentation

This folder contains project documentation, reports, and presentations.

## Contents

- **`reproducible_workflow.md`**: Guidelines for maintaining a reproducible research workflow
- Add your reports, presentations, and other documentation here

## Suggested Files to Add

- **Literature Review**: Summary of relevant academic papers and prior research
- **Data Documentation**: Detailed description of variables and data sources
- **Methodology Notes**: Technical details of analytical approaches
- **Meeting Notes**: Team meeting minutes and decisions
- **Final Report**: Complete capstone project report (R Markdown or LaTeX)
- **Presentation Slides**: Presentation materials for defense

## File Formats

- Use **R Markdown** (`.Rmd`) for dynamic reports that include code
- Use **Markdown** (`.md`) for static documentation
- Use **LaTeX** (`.tex`) for formal academic papers
- Use **PowerPoint/PDF** for presentations

Remember to commit documentation regularly to keep the team informed.
